Java hráči (počítač)

Po vstoupení do Minecraft menu klikněte na Multiplayer a dejte add server jako IP napište . Potom uložte a dejte jen play pro vstoupení do hry.

Bedrock hráči (mobil)

Po vstoupení do hry dejte hrát, potom servery a dejte přidat server. Server pojmenujte např. nordgate. Do adresa servru napište a do portu . Dejte uložit a můžete se připojit kliknutím na server.

Podporované verze Minecraftu

Java: až

Bedrock: až

Pro lepší zážitek ze hry, doporučujeme použivat nejnovějši verze, alespoň pro Javu nad verzi 1.16
